## 🎯 PLANO DE ESTUDO RECOMENDADO

### Dia 1: Aprendizado Básico (2 horas)

**Manhã (1 hora)**:
1. Leia a **Visão Geral** do README.md
2. Abra index.html, styles.css e scripts.js no VS Code
3. Veja a página rodando no navegador

**Tarde (1 hora)**:
1. Leia a **PARTE 1 (HTML)** do STUDY_GUIDE.md
2. Abra index.html e identifique os elementos mencionados
3. Use DevTools (F12) para inspecionar elementos

### Dia 2: Aprendizado de Estilos (2 horas)

**Manhã (1 hora)**:
1. Leia a **PARTE 2 (CSS)** do STUDY_GUIDE.md
2. Abra styles.css e localize cada seção (Sidebar, Main, Cards, etc)
3. Mude uma cor em :root e veja todas mudarem

**Tarde (1 hora)**:
1. Leia a **Sistema de Cores** do README.md
2. Experimente mudar cores no CSS
3. Teste a responsividade (F12 → Device Toggle)

### Dia 3: Aprendizado de Lógica (3 horas)

**Manhã (1.5 horas)**:
1. Leia a **PARTE 3 (JavaScript)** do STUDY_GUIDE.md (Seções 1-3)
2. Abra scripts.js e veja as seções mencionadas
3. Use console.log para entender dados

**Tarde (1.5 horas)**:
1. Continue lendo STUDY_GUIDE.md (Seções 4-6)
2. Trace um fluxo completo (usuário clica → função chamada → estado muda)
3. Teste em DevTools: `console.log(S)` para ver estado

### Dia 4: Preparação de Apresentação (1.5 horas)

**Manhã (45 min)**:
1. Leia todo o **PRESENTATION.md**
2. Identifique a estrutura de apresentação
3. Prepare suas notas/slides

**Tarde (45 min)**:
1. Faça uma apresentação de teste (sozinho ou com alguém)
2. Siga o roteiro de Demo
3. Cronometre para 5-15 minutos
